Single page application that shows artist images, names and station descriptions.

Application uses HTML, CSS and MeteorJS, a JavaScript reactive programming framework.

To install and run :

1. Install MeteorJS

  For Mac/Linux users : from terminal :
      curl https://install.meteor.com/ | sh

2. Packages intalled:

  meteor add iron:router
  meteor add http
  meteor add fourseven:sass

To run application:
  cd Responsive Artist
  meteor
